Create a new repository on the command line

touch README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/samear/Testing.git (HTTP )or git@github.com:sear/Testing.git (SSH)
git push -u origin master

Push an existing repository from the command line

git remote add origin https://github.com/samear/Testing.git (HTTP) or git@github.com:samear/Testing.git (SSH)
git push -u origin master

syncing files back and forward:
computer A (had the original git repo)
computer B (has the clode repo)

make some changes on computer A, then run
   git push origin master
go to computer B, then run
   git pull origin master

To sync your new changes, make some changes on computer B then push back

git push origin master
